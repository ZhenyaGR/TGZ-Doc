# –ö–æ–Ω—Å—Ç—Ä—É–∫—Ç–æ—Ä –°–æ–æ–±—â–µ–Ω–∏–π (Message Builder)

–ö–ª–∞—Å—Å `Message` –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ—Ç —É–¥–æ–±–Ω—ã–π "—Ç–µ–∫—É—á–∏–π –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å" (fluent interface) –¥–ª—è —Å–æ–∑–¥–∞–Ω–∏—è, —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏—è –∏ –æ—Ç–ø—Ä–∞–≤–∫–∏ —Å–æ–æ–±—â–µ–Ω–∏–π. –í–º–µ—Å—Ç–æ —Ç–æ–≥–æ —á—Ç–æ–±—ã —Å–æ–±–∏—Ä–∞—Ç—å –º–∞—Å—Å–∏–≤ –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–≤ –≤—Ä—É—á–Ω—É—é, –≤—ã –º–æ–∂–µ—Ç–µ –≤—ã—Å—Ç—Ä–∞–∏–≤–∞—Ç—å —Å–æ–æ–±—â–µ–Ω–∏–µ –≤ –≤–∏–¥–µ —Ü–µ–ø–æ—á–∫–∏ –≤—ã–∑–æ–≤–æ–≤, —á—Ç–æ –¥–µ–ª–∞–µ—Ç –∫–æ–¥ –±–æ–ª–µ–µ —á–∏—Å—Ç—ã–º –∏ —á–∏—Ç–∞–µ–º—ã–º.

## –û—Å–Ω–æ–≤—ã –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è

–û—Å–Ω–æ–≤–Ω–æ–π –ø—Ä–∏–Ω—Ü–∏–ø ‚Äî –≤—ã —Å–æ–∑–¥–∞–µ—Ç–µ —ç–∫–∑–µ–º–ø–ª—è—Ä —Å–æ–æ–±—â–µ–Ω–∏—è, –Ω–∞—Å—Ç—Ä–∞–∏–≤–∞–µ—Ç–µ –µ–≥–æ —Å –ø–æ–º–æ—â—å—é —Ä–∞–∑–ª–∏—á–Ω—ã—Ö –º–µ—Ç–æ–¥–æ–≤ –∏ –≤ –∫–æ–Ω—Ü–µ –≤—ã–∑—ã–≤–∞–µ—Ç–µ –º–µ—Ç–æ–¥ –æ—Ç–ø—Ä–∞–≤–∫–∏ (`send()`) –∏–ª–∏ —Ä–µ–¥–∞–∫—Ç–∏—Ä–æ–≤–∞–Ω–∏—è (`editText()`, `editCaption()` –∏ —Ç.–¥.).

### –ü—Ä–æ—Å—Ç–æ–µ —Ç–µ–∫—Å—Ç–æ–≤–æ–µ —Å–æ–æ–±—â–µ–Ω–∏–µ

–≠—Ç–æ —Å–∞–º—ã–π –±–∞–∑–æ–≤—ã–π –ø—Ä–∏–º–µ—Ä. –ú–µ—Ç–æ–¥ `$tg->message()` —è–≤–ª—è–µ—Ç—Å—è —Ñ–∞–±—Ä–∏–∫–æ–π, –∫–æ—Ç–æ—Ä–∞—è —Å–æ–∑–¥–∞–µ—Ç –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —ç–∫–∑–µ–º–ø–ª—è—Ä –∫–ª–∞—Å—Å–∞ `Message`.

```php
$bot->onCommand('start', function(TGZ $tg) {
    return $tg->message("–ü—Ä–∏–≤–µ—Ç, –º–∏—Ä!")->send();
});
```

### –¶–µ–ø–æ—á–∫–∞ –≤—ã–∑–æ–≤–æ–≤ (Chaining)

–°–∏–ª–∞ –∫–æ–Ω—Å—Ç—Ä—É–∫—Ç–æ—Ä–∞ –≤ –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç–∏ –æ–±—ä–µ–¥–∏–Ω—è—Ç—å –º–µ—Ç–æ–¥—ã –≤ —Ü–µ–ø–æ—á–∫—É. –ù–∞–ø—Ä–∏–º–µ—Ä, –æ—Ç–ø—Ä–∞–≤–∏–º —Ñ–æ—Ç–æ –≤ –æ—Ç–≤–µ—Ç –Ω–∞ —Å–æ–æ–±—â–µ–Ω–∏–µ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—è.

```php
$tg->message("–í–æ—Ç –≤–∞—à–µ —Ñ–æ—Ç–æ:")
   ->img('/path/to/image.jpg') // –î–æ–±–∞–≤–ª—è–µ–º –∏–∑–æ–±—Ä–∞–∂–µ–Ω–∏–µ
   ->reply()                  // –û—Ç–≤–µ—á–∞–µ–º –Ω–∞ —Ç–µ–∫—É—â–µ–µ —Å–æ–æ–±—â–µ–Ω–∏–µ
   ->send();                    // –û—Ç–ø—Ä–∞–≤–ª—è–µ–º
```

## –û—Å–Ω–æ–≤–Ω—ã–µ –º–µ—Ç–æ–¥—ã

### `text(string $text)`
–ó–∞–¥–∞–µ—Ç –∏–ª–∏ –∏–∑–º–µ–Ω—è–µ—Ç –æ—Å–Ω–æ–≤–Ω–æ–π —Ç–µ–∫—Å—Ç —Å–æ–æ–±—â–µ–Ω–∏—è –∏–ª–∏ –ø–æ–¥–ø–∏—Å—å –∫ –º–µ–¥–∏–∞.

```php
$tg->message("–ù–∞—á–∞–ª—å–Ω—ã–π —Ç–µ–∫—Å—Ç.")
   ->text("–¢–µ–∫—Å—Ç –∏–∑–º–µ–Ω–µ–Ω.") // –≠—Ç–æ—Ç —Ç–µ–∫—Å—Ç –±—É–¥–µ—Ç –æ—Ç–ø—Ä–∞–≤–ª–µ–Ω
   ->send();
```

### `parseMode(?string $mode)`
–£—Å—Ç–∞–Ω–∞–≤–ª–∏–≤–∞–µ—Ç —Ä–µ–∂–∏–º —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏—è —Ç–µ–∫—Å—Ç–∞.
*   **–î–æ—Å—Ç—É–ø–Ω—ã–µ —Ä–µ–∂–∏–º—ã:** `'HTML'`, `'MarkdownV2'`, `'Markdown'`.
*   –ï—Å–ª–∏ –ø–µ—Ä–µ–¥–∞—Ç—å `null` –∏–ª–∏ –ø—É—Å—Ç—É—é —Å—Ç—Ä–æ–∫—É, —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏–µ –±—É–¥–µ—Ç –æ—Ç–∫–ª—é—á–µ–Ω–æ.

```php
// –ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ HTML
$tg->message("<b>–ñ–∏—Ä–Ω—ã–π —Ç–µ–∫—Å—Ç</b> –∏ <i>–∫—É—Ä—Å–∏–≤</i>.")
   ->parseMode('HTML')
   ->send();

// –ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ MarkdownV2
$tg->message("*–ñ–∏—Ä–Ω—ã–π —Ç–µ–∫—Å—Ç* –∏ _–∫—É—Ä—Å–∏–≤_.")
   ->parseMode('MarkdownV2')
   ->send();
```

### `reply(?int $message_id = null)`
–î–µ–ª–∞–µ—Ç —Å–æ–æ–±—â–µ–Ω–∏–µ –æ—Ç–≤–µ—Ç–æ–º –Ω–∞ –¥—Ä—É–≥–æ–µ —Å–æ–æ–±—â–µ–Ω–∏–µ.
*   –ï—Å–ª–∏ `$message_id` –Ω–µ —É–∫–∞–∑–∞–Ω, –æ—Ç–≤–µ—Ç –±—É–¥–µ—Ç –Ω–∞ —Ç–µ–∫—É—â–µ–µ —Å–æ–æ–±—â–µ–Ω–∏–µ –∏–∑ –∫–æ–Ω—Ç–µ–∫—Å—Ç–∞.

```php
// –û—Ç–≤–µ—Ç –Ω–∞ —Ç–µ–∫—É—â–µ–µ —Å–æ–æ–±—â–µ–Ω–∏–µ
$tg->message("–≠—Ç–æ –æ—Ç–≤–µ—Ç.")->reply()->send();

// –û—Ç–≤–µ—Ç –Ω–∞ –∫–æ–Ω–∫—Ä–µ—Ç–Ω–æ–µ —Å–æ–æ–±—â–µ–Ω–∏–µ
$tg->message("–û—Ç–≤–µ—á–∞—é –Ω–∞ —Å–æ–æ–±—â–µ–Ω–∏–µ 12345.")->reply(12345)->send();
```

## –†–∞–±–æ—Ç–∞ —Å –º–µ–¥–∏–∞

–ö–æ–Ω—Å—Ç—Ä—É–∫—Ç–æ—Ä –ø–æ–∑–≤–æ–ª—è–µ—Ç –ª–µ–≥–∫–æ –æ—Ç–ø—Ä–∞–≤–ª—è—Ç—å —Ä–∞–∑–ª–∏—á–Ω—ã–µ —Ç–∏–ø—ã –º–µ–¥–∏–∞. –í –∫–∞—á–µ—Å—Ç–≤–µ –∏—Å—Ç–æ—á–Ω–∏–∫–∞ —Ñ–∞–π–ª–∞ –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å:
1.  **–õ–æ–∫–∞–ª—å–Ω—ã–π –ø—É—Ç—å:** `/path/to/file.jpg`
2.  **URL:** `https://example.com/image.png`
3.  **Telegram `file_id`:** `AgACAgIAAxkBAA...`

### –û—Ç–ø—Ä–∞–≤–∫–∞ –æ–¥–Ω–æ–≥–æ –º–µ–¥–∏–∞—Ñ–∞–π–ª–∞

```php
// –û—Ç–ø—Ä–∞–≤–∫–∞ —Ñ–æ—Ç–æ –ø–æ URL
$tg->message("–§–æ—Ç–æ –∏–∑ –∏–Ω—Ç–µ—Ä–Ω–µ—Ç–∞")
   ->img('https://placehold.co/600x400')
   ->send();

// –û—Ç–ø—Ä–∞–≤–∫–∞ –≤–∏–¥–µ–æ —Å –¥–∏—Å–∫–∞
$tg->message("–í–∏–¥–µ–æ —Å –¥–∏—Å–∫–∞")
   ->video('/data/my_video.mp4')
   ->send();

// –û—Ç–ø—Ä–∞–≤–∫–∞ –¥–æ–∫—É–º–µ–Ω—Ç–∞ –ø–æ file_id
$tg->message("–î–æ–∫—É–º–µ–Ω—Ç –ø–æ ID")
   ->doc('BQACAgIAAxkBAAE..._')
   ->send();
```

### –û—Ç–ø—Ä–∞–≤–∫–∞ –≥—Ä—É–ø–ø—ã –º–µ–¥–∏–∞ (Media Group)

–ß—Ç–æ–±—ã –æ—Ç–ø—Ä–∞–≤–∏—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ —Ñ–æ—Ç–æ –∏–ª–∏ –≤–∏–¥–µ–æ –≤ –æ–¥–Ω–æ–º —Å–æ–æ–±—â–µ–Ω–∏–∏ (–∞–ª—å–±–æ–º–æ–º), –ø–µ—Ä–µ–¥–∞–π—Ç–µ –≤ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–∏–π –º–µ—Ç–æ–¥ –º–∞—Å—Å–∏–≤ –∏—Å—Ç–æ—á–Ω–∏–∫–æ–≤.

> **–í–∞–∂–Ω–æ:** –ü–æ–¥–ø–∏—Å—å (`text()`) –±—É–¥–µ—Ç –¥–æ–±–∞–≤–ª–µ–Ω–∞ —Ç–æ–ª—å–∫–æ –∫ –ø–µ—Ä–≤–æ–º—É —ç–ª–µ–º–µ–Ω—Ç—É –≤ –º–µ–¥–∏–∞–≥—Ä—É–ø–ø–µ. –í –æ–¥–Ω–æ–π –≥—Ä—É–ø–ø–µ –º–æ–∂–Ω–æ —Å–º–µ—à–∏–≤–∞—Ç—å —Ñ–æ—Ç–æ –∏ –≤–∏–¥–µ–æ. –î–æ–∫—É–º–µ–Ω—Ç—ã –∏ –∞—É–¥–∏–æ—Ñ–∞–π–ª—ã –º–æ–∂–Ω–æ –≥—Ä—É–ø–ø–∏—Ä–æ–≤–∞—Ç—å —Ç–æ–ª—å–∫–æ —Å —Ñ–∞–π–ª–∞–º–∏ —Ç–æ–≥–æ –∂–µ —Ç–∏–ø–∞.

```php
$photos = [
    'https://placehold.co/600x400/EEE/31343C',
    '/path/to/local/image.jpg',
    'AgACAgIAAxkBAAE...' // file_id

$tg->message("–í–æ—Ç –∞–ª—å–±–æ–º –∏–∑ —Ç—Ä–µ—Ö —Ñ–æ—Ç–æ")
   ->img($photos)
   ->send();
```

### `mediaPreview(string $url)`
–≠—Ç–æ—Ç –º–µ—Ç–æ–¥ –ø–æ–∑–≤–æ–ª—è–µ—Ç –¥–æ–±–∞–≤–∏—Ç—å –∫ —Å–æ–æ–±—â–µ–Ω–∏—é –ø—Ä–µ–¥–ø—Ä–æ—Å–º–æ—Ç—Ä —Å—Å—ã–ª–∫–∏ (rich preview), –Ω–µ –æ—Ç–æ–±—Ä–∞–∂–∞—è —Å–∞–º—É —Å—Å—ã–ª–∫—É –≤ —Ç–µ–∫—Å—Ç–µ. –≠—Ç–æ –¥–æ—Å—Ç–∏–≥–∞–µ—Ç—Å—è –∑–∞ —Å—á–µ—Ç –¥–æ–±–∞–≤–ª–µ–Ω–∏—è –Ω–µ–≤–∏–¥–∏–º–æ–π —Å—Å—ã–ª–∫–∏ –≤ –Ω–∞—á–∞–ª–æ —Ç–µ–∫—Å—Ç–∞.

```php
$tg->message("–≠—Ç–∞ —Å—Ç–∞—Ç—å—è –æ—á–µ–Ω—å –ø–æ–ª–µ–∑–Ω–∞.")
   ->mediaPreview('https://example.com/some-article')
   ->parseMode('HTML') // –†–µ–∂–∏–º –ø–∞—Ä—Å–∏–Ω–≥–∞ –≤–∞–∂–µ–Ω –¥–ª—è –∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ–π —Ä–∞–±–æ—Ç—ã
   ->send();
```

## –ö–ª–∞–≤–∏–∞—Ç—É—Ä—ã

–ú–µ—Ç–æ–¥ `kbd()` –ø–æ–∑–≤–æ–ª—è–µ—Ç —É–ø—Ä–∞–≤–ª—è—Ç—å –∫–∞–∫ –æ–±—ã—á–Ω—ã–º–∏ (`reply`), —Ç–∞–∫ –∏ –≤—Å—Ç—Ä–æ–µ–Ω–Ω—ã–º–∏ (`inline`) –∫–ª–∞–≤–∏–∞—Ç—É—Ä–∞–º–∏.

### Reply-–∫–ª–∞–≤–∏–∞—Ç—É—Ä–∞

```php

    [['text' => '–ö–Ω–æ–ø–∫–∞ 1'], ['text' => '–ö–Ω–æ–ø–∫–∞ 2']],
    [['text' => '–ó–∞–ø—Ä–æ—Å–∏—Ç—å –∫–æ–Ω—Ç–∞–∫—Ç', 'request_contact' => true]],
];

$tg->message("–í—ã–±–µ—Ä–∏—Ç–µ –æ–ø—Ü–∏—é:")
   ->kbd($buttons, resize_keyboard: true, one_time_keyboard: true)
   ->send();
```

### Inline-–∫–ª–∞–≤–∏–∞—Ç—É—Ä–∞

```php
$buttons = [
    [['text' => 'Google', 'url' => 'https://google.com']],
    [['text' => '–ù–∞–∂–º–∏ –º–µ–Ω—è', 'callback_data' => 'button_pressed']],
];

$tg->message("–≠—Ç–æ inline-–∫–ª–∞–≤–∏–∞—Ç—É—Ä–∞:")
   ->kbd($buttons, inline: true)
   ->send();
```

### –£–¥–∞–ª–µ–Ω–∏–µ –∫–ª–∞–≤–∏–∞—Ç—É—Ä—ã

–ß—Ç–æ–±—ã —É–±—Ä–∞—Ç—å reply-–∫–ª–∞–≤–∏–∞—Ç—É—Ä—É —É –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—è, –≤—ã–∑–æ–≤–∏—Ç–µ `kbd()` —Å –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–º `remove_keyboard: true`.

## –†–µ–¥–∞–∫—Ç–∏—Ä–æ–≤–∞–Ω–∏–µ —Å–æ–æ–±—â–µ–Ω–∏–π
