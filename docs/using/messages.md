# –ö–æ–Ω—Å—Ç—Ä—É–∫—Ç–æ—Ä –°–æ–æ–±—â–µ–Ω–∏–π (Message Builder)

–ö–ª–∞—Å—Å `Message` –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ—Ç —É–¥–æ–±–Ω—ã–π "—Ç–µ–∫—É—á–∏–π –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å" (fluent interface) –¥–ª—è —Å–æ–∑–¥–∞–Ω–∏—è, —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏—è –∏ –æ—Ç–ø—Ä–∞–≤–∫–∏ —Å–æ–æ–±—â–µ–Ω–∏–π. –í–º–µ—Å—Ç–æ —Ç–æ–≥–æ —á—Ç–æ–±—ã —Å–æ–±–∏—Ä–∞—Ç—å –º–∞—Å—Å–∏–≤ –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–≤ –≤—Ä—É—á–Ω—É—é, –≤—ã –º–æ–∂–µ—Ç–µ –≤—ã—Å—Ç—Ä–∞–∏–≤–∞—Ç—å —Å–æ–æ–±—â–µ–Ω–∏–µ –≤ –≤–∏–¥–µ —Ü–µ–ø–æ—á–∫–∏ –≤—ã–∑–æ–≤–æ–≤, —á—Ç–æ –¥–µ–ª–∞–µ—Ç –∫–æ–¥ –±–æ–ª–µ–µ —á–∏—Å—Ç—ã–º –∏ —á–∏—Ç–∞–µ–º—ã–º.

## –û—Å–Ω–æ–≤—ã –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è

–û—Å–Ω–æ–≤–Ω–æ–π –ø—Ä–∏–Ω—Ü–∏–ø ‚Äî –≤—ã —Å–æ–∑–¥–∞–µ—Ç–µ —ç–∫–∑–µ–º–ø–ª—è—Ä —Å–æ–æ–±—â–µ–Ω–∏—è, –Ω–∞—Å—Ç—Ä–∞–∏–≤–∞–µ—Ç–µ –µ–≥–æ —Å –ø–æ–º–æ—â—å—é —Ä–∞–∑–ª–∏—á–Ω—ã—Ö –º–µ—Ç–æ–¥–æ–≤ –∏ –≤ –∫–æ–Ω—Ü–µ –≤—ã–∑—ã–≤–∞–µ—Ç–µ –º–µ—Ç–æ–¥ –æ—Ç–ø—Ä–∞–≤–∫–∏ (`send()`) –∏–ª–∏ —Ä–µ–¥–∞–∫—Ç–∏—Ä–æ–≤–∞–Ω–∏—è (`editText()`, `editCaption()` –∏ —Ç.–¥.).

### –ü—Ä–æ—Å—Ç–æ–µ —Ç–µ–∫—Å—Ç–æ–≤–æ–µ —Å–æ–æ–±—â–µ–Ω–∏–µ

–≠—Ç–æ —Å–∞–º—ã–π –±–∞–∑–æ–≤—ã–π –ø—Ä–∏–º–µ—Ä. –ú–µ—Ç–æ–¥ `$tg->msg()` —è–≤–ª—è–µ—Ç—Å—è —Ñ–∞–±—Ä–∏–∫–æ–π, –∫–æ—Ç–æ—Ä–∞—è —Å–æ–∑–¥–∞–µ—Ç –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —ç–∫–∑–µ–º–ø–ª—è—Ä –∫–ª–∞—Å—Å–∞ `Message`.

```php

    $tg->msg("–ü—Ä–∏–≤–µ—Ç, –º–∏—Ä!")->send();

```

### –¶–µ–ø–æ—á–∫–∞ –≤—ã–∑–æ–≤–æ–≤ (Chaining)

–°–∏–ª–∞ –∫–æ–Ω—Å—Ç—Ä—É–∫—Ç–æ—Ä–∞ –≤ –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç–∏ –æ–±—ä–µ–¥–∏–Ω—è—Ç—å –º–µ—Ç–æ–¥—ã –≤ —Ü–µ–ø–æ—á–∫—É. –ù–∞–ø—Ä–∏–º–µ—Ä, –æ—Ç–ø—Ä–∞–≤–∏–º —Ñ–æ—Ç–æ –≤ –æ—Ç–≤–µ—Ç –Ω–∞ —Å–æ–æ–±—â–µ–Ω–∏–µ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—è.

```php
   ->img('/path/to/image.jpg')  // –î–æ–±–∞–≤–ª—è–µ–º –∏–∑–æ–±—Ä–∞–∂–µ–Ω–∏–µ
   ->reply()                    // –û—Ç–≤–µ—á–∞–µ–º –Ω–∞ —Ç–µ–∫—É—â–µ–µ —Å–æ–æ–±—â–µ–Ω–∏–µ
   ->send();                    // –û—Ç–ø—Ä–∞–≤–ª—è–µ–º
```

## –û—Å–Ω–æ–≤–Ω—ã–µ –º–µ—Ç–æ–¥—ã

### `parseMode(?string $mode)`
–£—Å—Ç–∞–Ω–∞–≤–ª–∏–≤–∞–µ—Ç —Ä–µ–∂–∏–º —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏—è —Ç–µ–∫—Å—Ç–∞.

### `reply(?int $message_id = null)`

### `action(?string $action = 'typing')`

–≠—Ç–æ—Ç –º–µ—Ç–æ–¥ –æ—Ç–ø—Ä–∞–≤–ª—è–µ—Ç –≤ —á–∞—Ç —Å—Ç–∞—Ç—É—Å –¥–µ–π—Å—Ç–≤–∏—è, —á—Ç–æ–±—ã –ø–æ–∫–∞–∑–∞—Ç—å –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—é, —á—Ç–æ –±–æ—Ç –∑–∞–Ω—è—Ç –∫–∞–∫–æ–π-—Ç–æ –∑–∞–¥–∞—á–µ–π (–Ω–∞–ø—Ä–∏–º–µ—Ä, '–ø–µ—á–∞—Ç–∞–µ—Ç...', '–∑–∞–≥—Ä—É–∂–∞–µ—Ç —Ñ–æ—Ç–æ...'). –≠—Ç–æ —É–ª—É—á—à–∞–µ—Ç –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–π –æ–ø—ã—Ç, —Ç–∞–∫ –∫–∞–∫ –¥–∞–µ—Ç –ø–æ–Ω—è—Ç—å, —á—Ç–æ –±–æ—Ç –Ω–µ –∑–∞–≤–∏—Å, –∞ –≤—ã–ø–æ–ª–Ω—è–µ—Ç –æ–ø–µ—Ä–∞—Ü–∏—é.

**–ö–ª—é—á–µ–≤–∞—è –æ—Å–æ–±–µ–Ω–Ω–æ—Å—Ç—å:** –≤ –æ—Ç–ª–∏—á–∏–µ –æ—Ç –¥—Ä—É–≥–∏—Ö –º–µ—Ç–æ–¥–æ–≤ –∫–æ–Ω—Å—Ç—Ä—É–∫—Ç–æ—Ä–∞, `action()` –æ—Ç–ø—Ä–∞–≤–ª—è–µ—Ç –∑–∞–ø—Ä–æ—Å –∫ Telegram API **–Ω–µ–º–µ–¥–ª–µ–Ω–Ω–æ** –≤ –º–æ–º–µ–Ω—Ç —Å–≤–æ–µ–≥–æ –≤—ã–∑–æ–≤–∞, –∞ –Ω–µ –ø—Ä–∏ –≤—ã–∑–æ–≤–µ `send()`. –¢–µ–º –Ω–µ –º–µ–Ω–µ–µ, –æ–Ω –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —ç–∫–∑–µ–º–ø–ª—è—Ä `Message`, –ø–æ–∑–≤–æ–ª—è—è –ø—Ä–æ–¥–æ–ª–∂–∞—Ç—å —Ü–µ–ø–æ—á–∫—É –≤—ã–∑–æ–≤–æ–≤.


#### –î–æ—Å—Ç—É–ø–Ω—ã–µ –¥–µ–π—Å—Ç–≤–∏—è

–ö–æ–Ω—Å—Ç—Ä—É–∫—Ç–æ—Ä –ø–æ–∑–≤–æ–ª—è–µ—Ç –ª–µ–≥–∫–æ –æ—Ç–ø—Ä–∞–≤–ª—è—Ç—å —Ä–∞–∑–ª–∏—á–Ω—ã–µ —Ç–∏–ø—ã –º–µ–¥–∏–∞. –í –∫–∞—á–µ—Å—Ç–≤–µ –∏—Å—Ç–æ—á–Ω–∏–∫–∞ —Ñ–∞–π–ª–∞ –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å:

### –û—Ç–ø—Ä–∞–≤–∫–∞ –≥—Ä—É–ø–ø—ã –º–µ–¥–∏–∞ (Media Group)

## –ö–ª–∞–≤–∏–∞—Ç—É—Ä—ã

–ú–µ—Ç–æ–¥ `kbd()` –ø–æ–∑–≤–æ–ª—è–µ—Ç —É–ø—Ä–∞–≤–ª—è—Ç—å –∫–∞–∫ –æ–±—ã—á–Ω—ã–º–∏ (`reply`), —Ç–∞–∫ –∏ –≤—Å—Ç—Ä–æ–µ–Ω–Ω—ã–º–∏ (`inline`) –∫–ª–∞–≤–∏–∞—Ç—É—Ä–∞–º–∏.

## –†–µ–¥–∞–∫—Ç–∏—Ä–æ–≤–∞–Ω–∏–µ —Å–æ–æ–±—â–µ–Ω–∏–π
