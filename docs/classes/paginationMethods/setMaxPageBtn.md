# setMaxPageBtn
Метод устанавливает максимальное количество номерных кнопок-страниц.

## Параметры метода
| # | Название |  Тип  |                     Описание                     |
|:-:|:--------:|:-----:|:------------------------------------------------:|
| 1 | **max**  | `int` | Максимальное количество кнопок (По умолчанию: 5) |

## Возвращает
`Pagination` — экземпляр класса Pagination

## Пример использования
```php
$tg->pagination()->setMaxPageBtn(5);
```
