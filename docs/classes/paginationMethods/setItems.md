# setItems
Метод устанавливает массив элементов (кнопок), которые необходимо разбить на страницы.

## Параметры метода
| # | Название  |   Тип   |             Описание             |
|:-:|:---------:|:-------:|:--------------------------------:|
| 1 | **items** | `array` | Массив кнопок (callback или url) |

## Возвращает
`Pagination` — экземпляр класса Pagination

## Пример использования
```php
$items = [
    Button::cb('Кнопка 1', 'btn_1'),
    Button::cb('Кнопка 2', 'btn_2'),
    // ...
];

$tg->pagination()->setItems($items);
```
