# setPerPage
Опциональный метод, устанавливает максимальное количество элементов (кнопок) на одной странице.

## Параметры метода
| # |  Название   |  Тип  |               Описание                |
|:-:|:-----------:|:-----:|:-------------------------------------:|
| 1 | **perPage** | `int` | Количество элементов (по умолчанию 5) |

## Возвращает
`Pagination` — экземпляр класса Pagination

## Пример использования
```php
$tg->pagination()->setPerPage(10);
```
