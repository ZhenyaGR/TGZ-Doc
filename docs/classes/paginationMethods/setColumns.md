# setColumns
Опциональный метод, устанавливает количество колонок для отображения элементов (кнопок) на странице.

## Параметры метода
| # |  Название   |  Тип  |              Описание               |
|:-:|:-----------:|:-----:|:-----------------------------------:|
| 1 | **columns** | `int` | Количество колонок (по умолчанию 1) |

## Возвращает
`Pagination` — экземпляр класса Pagination

## Пример использования
```php
// Элементы будут располагаться по 2 в ряд
$tg->pagination()->setColumns(2);
```
