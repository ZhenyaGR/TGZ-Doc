# setSigns
Опциональный метод, позволяет изменить текст на кнопках "Предыдущая страница" и "Следующая страница".

## Параметры метода
| # |  Название  |   Тип    |           Описание            |
|:-:|:----------:|:--------:|:-----------------------------:|
| 1 | **prevText** | `string` | Текст кнопки "Назад" (по умолчанию `<`) |
| 2 | **nextText** | `string` | Текст кнопки "Вперед" (по умолчанию `>`) |

## Возвращает
`Pagination` — экземпляр класса Pagination

## Пример использования
```php
$tg->pagination()->setSigns('⬅️ Назад', 'Вперед ➡️');
```
