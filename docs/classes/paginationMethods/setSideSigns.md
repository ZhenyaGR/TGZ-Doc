# setSideSigns
Опциональный метод, включает отображение кнопок быстрой навигации ("В начало" и "В конец") и задает их текст.

## Параметры метода
| # |   Название    |   Тип    |        Описание         |
|:-:|:-------------:|:--------:|:-----------------------:|
| 1 | **firstText** | `string` | Текст кнопки "В начало" |
| 2 | **lastText**  | `string` | Текст кнопки "В конец"  |

## Возвращает
`Pagination` — экземпляр класса Pagination

## Пример использования
```php
$tg->pagination()->setSideSigns('⏪', '⏩');
```
