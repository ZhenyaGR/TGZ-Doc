# create
Метод собирает и возвращает готовый массив клавиатуры с учетом текущей страницы, настроек отображения и навигации.

## Параметры метода
Метод не принимает параметров.

## Возвращает
`array` — Массив кнопок, готовый для передачи в методы `kbd()` или `inlineKbd()`.

## Пример использования
```php
$keyboard = $tg->pagination()
    ->setItems($items)
    ->setPage(1)
    ->setPrefix('p_')
    ->create();

$tg->msg('Страница 1')->inlineKbd($keyboard)->send();
```
