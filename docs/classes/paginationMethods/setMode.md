# setMode
Опциональный метод, устанавливает вид кнопок навигации 

Стрелки: `<` `>`

Номера: `1`, `2`, `3`

::: warning Вид взаимоисключающие
Если вы устанавливаете вид кнопок "Стрелки", методы для "Номеров" работать не будут.
:::

## Параметры метода
| # | Название |       Тип        |             Описание             |
|:-:|:--------:|:----------------:|:--------------------------------:|
| 1 | **mode** | `PaginationMode` | Константа типа кнопок (см. ниже) |

### Возможные значения (Константы)
PaginationMode — перечисление `(Enum)`, определяющее вид кнопок навигации.

* `PaginationMode::ARROWS` (0) — Стандартные стрелки навигации "Предыдущая страница" и "Следующая страница" (по умолчанию).
* `PaginationMode::NUMBERS` (1) — Несколько номеров страниц на строке.

## Возвращает
`Pagination` — экземпляр класса Pagination

## Пример использования
```php
use ZhenyaGR\TGZ\Pagination;

// Пример 1: Стрелки
$tg->pagination()->setMode(PaginationMode::ARROWS);

// Пример 2: Номера страниц
$tg->pagination()->setMode(PaginationMode::NUMBERS);
```
