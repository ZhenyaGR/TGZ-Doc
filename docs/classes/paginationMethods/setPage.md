# setPage
Опциональный метод, устанавливает номер текущей страницы, которую нужно отобразить.

## Параметры метода
| # | Название |  Тип  |           Описание           |
|:-:|:--------:|:-----:|:----------------------------:|
| 1 | **page** | `int` | Номер страницы (начиная с 1) |

## Возвращает
`Pagination` — экземпляр класса Pagination

## Пример использования
```php
// Обычно номер страницы берётся из пришедшего callback_data (например, 'page_2')
$currentPage = 2; 
$tg->pagination()->setPage($currentPage);
```
