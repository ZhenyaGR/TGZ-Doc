# session
Универсальный метод для работы с данными пользователя в рамках диалога. Позволяет как сохранять промежуточные данные (например, имя, возраст), так и получать их.

## Параметры метода
| # | Название |       Тип       |                                                    Описание                                                    |
|:-:|:--------:|:---------------:|:--------------------------------------------------------------------------------------------------------------:|
| 1 | **data** | `array`\|`null` | Если передан массив — данные сохраняются (объединяются с текущими). Если `null` — возвращаются текущие данные. |

## Возвращает
`array` — Текущий массив данных сессии.

## Пример использования
```php
// Сохранение данных
$tg->session(['age' => 25]);

// Получение данных
$data = $tg->session();
$age = $data['age']; // 25
```
