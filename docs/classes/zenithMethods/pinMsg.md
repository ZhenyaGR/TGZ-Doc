# pinMsg
Метод закрепляет указанное сообщение в чате.

## Параметры метода
| # |         Название         |         Тип         | Описание                                                                   |
|:-:|:------------------------:|:-------------------:|:---------------------------------------------------------------------------|
| 1 |        **msg_id**        |     `int\|null`     | ID сообщения, которое нужно закрепить. Если `null`, берется из контекста.  |
| 2 |       **chat_id**        | `int\|string\|null` | ID чата. По умолчанию - текущий чат.                                       |
| 3 | **disable_notification** |       `bool`        | Если `true`, уведомление о закрепе придет без звука. По умолчанию `false`. |

## Возвращает
`array` - массив с ответом от Telegram.

## Примеры использования

### 1. Закрепить текущее сообщение
```php
$tg->pinMsg();
```

### 2. Закрепить без уведомления (тихо)
```php
$tg->pinMsg(disable_notification: true);
```

### 3. Закрепить конкретное сообщение в другом чате
```php
$tg->pinMsg(
    msg_id: 12345,
    chat_id: '@my_channel',
    disable_notification: true
);
```
