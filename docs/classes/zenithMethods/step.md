# step
Метод устанавливает новое состояние для текущего пользователя. Используется внутри обработчиков для продвижения по диалогу.

## Параметры метода
| # | Название  |   Тип    |             Описание              |
|:-:|:---------:|:--------:|:---------------------------------:|
| 1 | **state** | `string` | Название нового состояния (шага). |

## Возвращает
`void`

## Пример использования
```php
$bot->onCommand('start', '/start')->func(function(ZG $tg) {
    $tg->reply('Как вас зовут?');
    $tg->step('wait_name'); // Переход в состояние ожидания имени
});
```
