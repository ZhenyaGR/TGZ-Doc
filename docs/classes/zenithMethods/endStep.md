# endStep
Метод сбрасывает текущее состояние пользователя, завершая диалог.

## Параметры метода
| # |    Название    |  Тип   |                           Описание                            |
|:-:|:--------------:|:------:|:-------------------------------------------------------------:|
| 1 | **clear_data** | `bool` | Удалять данные сессии? `True`/`False`. \ По умолчанию: `true` |

## Возвращает
`void`

## Пример использования
```php
$bot->onState('final_step')->func(function(ZG $tg) {
    $tg->msg('Спасибо, опрос завершен!')->send();
    $tg->endStep(); // Пользователь больше не находится в состоянии
    // Так же будут удалены все накопленные данные  
});
```
