# getStorage
Метод возвращает текущий объект хранилища для прямых манипуляций (например, если нужно вызвать специфичный метод хранилища).

## Возвращает
`StorageInterface`|`null` — Объект хранилища или null, если оно не задано.

## Пример использования
```php
$data = $tg->getStorage()->getSessionData($user_id);
```
