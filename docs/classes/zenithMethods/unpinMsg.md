# unpinMsg
Метод открепляет одно сообщение или сразу все сообщения в чате.

## Параметры метода
| # |  Название   |         Тип         | Описание                                                                      |
|:-:|:-----------:|:-------------------:|:------------------------------------------------------------------------------|
| 1 | **msg_id**  |     `int\|null`     | ID сообщения, которое нужно открепить. Если `null`, берется из контекста.     |
| 2 | **chat_id** | `int\|string\|null` | ID чата. По умолчанию - текущий чат.                                          |
| 3 |   **all**   |       `bool`        | Если `true`, будут откреплены **все** сообщения в чате. По умолчанию `false`. |

## Возвращает
`array` - массив с ответом от Telegram.

## Примеры использования

### 1. Открепить текущее сообщение
```php
$tg->unpinMsg();
```

### 2. Открепить конкретное сообщение
```php
$tg->unpinMsg(msg_id: 12345);
```

### 3. Открепить ВСЕ сообщения в чате
```php
$tg->unpinMsg(all: true);
```
