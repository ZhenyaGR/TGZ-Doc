## msg
Метод `msg` является «синтаксическим сахаром», который используется для создания и экземпляра класса `Message`. Он предназначен для создания сообщений и отправки их в ботах.
 
### Параметры метода
| # | Название  |   Тип    | Описание                                              |
|:-:|:---------:|:--------:|-------------------------------------------------------|
| 1 | **text** | `string`\|`null` | Текст желаемого сообщения |

### Возвращает
`Message` - экземпляр класса `Message`.

### Пример использования
```php
<?php
require_once __DIR__ . 'vendor/autoload.php';
use ZhenyaGR\TGZ\TGZ;

$tg = TGZ::create('ТОКЕН');

$tg->msg("Привет, Мир")->send();
```
